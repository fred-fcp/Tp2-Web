Criterio de selección
Todas las imágenes comparten una paleta cálida oscura: marrones, tierras, negros con sutiles reflejos de luz natural. Evitar imágenes con dominante fría, blanca o saturada — rompen la coherencia cromática del sistema.
Reglas de uso
Las fotos nunca se muestran crudas — siempre llevan un overlay de gradiente oscuro inferior o un componente glass encima. Las proporciones de las cards (4:3 horizontal, 3:4 vertical) deben respetarse para no distorsionar el encuadre original.
